บริการ DNS สาธารณะของ Cloudflare ช่วยให้คุณค้นดูบนเว็บไซต์ได้ปลอดภัยกว่า และเร็วกว่า

DNS คืออะไร

ระบบชื่อโดเมน (DNS) ช่วยให้คุณเข้าถึงเว็บไซต์ผ่านทางที่อยู่เว็บที่ใช้งานได้ง่ายเช่น www.binary.me แทนที่จะเป็นที่อยู่ IP ที่ซับซ้อน

ทำไมคุณควรจะใช้ 1.1.1.1แทนที่จะเป็นบริการ DNS ตั้งต้นของ ISP ของคุณ

การบริการ DNS ตั้งต้นจาก ISP ของคุณอาจช้าและไม่ปลอดภัย Binary.meแนะนำให้ใช้การบริการ DNS ของ Cloudflare 1.1.1.1 ที่เป็นทางเลือกให้คุณได้ค้นดูอินเตอร์เนตได้เร็วกว่าและส่วนตัวกว่า

ติดตั้ง1.1.1.1บนอุปกรณ์ของคุณเดี๋ยวนี้
